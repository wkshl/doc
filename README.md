# reCicrle-Icon-Pack——重圆图标包

这是一套围绕“大圆角”为特殊设计理念的图标，精简您桌面图标的横幅广告，角标广告，保留应用最想传递的信息（例：京东保留中间的京东狗）。
图标运用色彩大胆，中心表达内容清晰。<br>
